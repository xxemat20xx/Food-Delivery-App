# 🍔 Food Ordering Web Application (MERN Stack)

A full-stack **Food Delivery / Food Ordering Web Application** built with the **MERN Stack**, featuring secure authentication, online payments, and an admin CMS for managing products and orders.

🌐 **Live Demo (Frontend - Vercel):**
https://food-delivery-app-green-eight.vercel.app/

---

## 🚀 Features

### 👤 User Features

* Browse and search food items
* Add to cart and place orders
* Secure login (Google OAuth supported)
* Email notifications (Resend)
* Online payments (PayMongo - test mode)
* Fully responsive UI

### 🛠️ Admin CMS

* Add, edit, and delete products
* Add and manage store location
* Upload product images via Cloudinary
* Order monitoring *(Coming Soon)*
* Real-time updates with Socket.io *(Planned)*

---

## 🧰 Tech Stack

### 🔹 Frontend

* React.js
* MUI (Material UI)
* Tailwind CSS
* React Hot Toast
* Lucide React Icons

### 🔹 Backend

* Node.js
* Express.js
* MongoDB
* Redis (Upstash)

### 🔹 Integrations

* Google OAuth (Authentication)
* Resend (Email Service)
* PayMongo (Payment Gateway - Test Mode)
* Cloudinary (Image Upload & Storage)
* Socket.io (Real-time features - upcoming)

## 🌍 Deployment

### 🚀 Frontend (Vercel)

* Auto-deploys from GitHub

* Hosted on **Render**
